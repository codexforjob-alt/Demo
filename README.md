# Emil Stom — demo concept

Limited, unofficial one-page website concept for the public Instagram profile
`@emil_stom_kz`.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Demo safeguards

- The page is visibly labelled as a demo concept.
- Appointment controls do not transmit or store data.
- Real phone numbers, exact addresses, prices, reviews and patient photos are omitted.
- Search indexing is disabled through metadata and `public/robots.txt`.
- The footer states that this is not the clinic's official website.

This is a limited demo concept, not the production website.
